# 🌦️ Weather App | Weather Forecast Dashboard

A sleek, responsive, and dynamic weather application built with Vanilla JavaScript, Bootstrap, and WeatherAPI. This app provides real-time weather updates and a 3-day forecast with a dynamic UI that changes based on the current weather conditions.

## 🚀 Live Demo
[View Live Demo](https://omarhassan1911b-wq.github.io/Weather-App/) 

## ✨ Features

* **Real-time Search:** Get weather information for any city worldwide instantly as you type.
* **3-Day Forecast:** Displays current day details plus the next two days (High/Low temperatures).
* **Dynamic Backgrounds:** Background images change automatically based on weather conditions (Sunny, Rain, Snow, Clouds, etc.).
* **Smooth Animations:** Custom CSS transitions and `Animate.css` for a premium feel.
* **Responsive Design:** Fully optimized for all screen sizes using Bootstrap 5.
* **Detailed Metrics:** Shows humidity, wind speed, and wind direction.

## 🛠️ Built With

* **HTML5 & CSS3:** Semantic structure and custom styling.
* **JavaScript (ES6+):** Asynchronous API calls (Fetch) and DOM manipulation.
* **Bootstrap 5:** Grid system and responsive layout.
* **WeatherAPI:** Real-time meteorological data.
* **FontAwesome:** Intuitive weather and social media icons.
